# Homemade Web App

A simple high-school web application built using PHP, HTML, CSS, and MySQL.  
This project demonstrates basic web pages, forms, database interactions, and images.

## Features
- Multiple web pages: Home, About, Contact, Trade, Product pages
- User signup and signin functionality
- Database operations: insert and delete entries
- CSS styling for layout and design
- Images for products and interface

## Current Files
- PHP files for logic and forms:
  - `contact.php`, `form.php`, `insert.php`, `rdelete.php`, `signin.php`, `signin_database_tally.php`, `signup.php`, `trade.php`
- HTML pages:
  - `about.html`, `index.html`, `product.html`, `rdelete_entry.html`, `trade.html`
- CSS files:
  - `style.css`, `styles.css`
- Images:
  - `bg.jpg`, `homemade.png`, `insta.png`, `product1.jpg`, `product2.jpg`, `product3.jpg`, `product4.jpg`, `product5.jpg`, `product6.jpg`
- Special file:
  - `CNAME`

## How to Run
1. Copy the `homemade` folder into your XAMPP `htdocs` directory.
2. Start **Apache** and **MySQL** from XAMPP.
3. Open the project in a browser:
http://localhost/homemade/index.html

(or `index.php` depending on which file is the homepage)

## Notes
- This is a **sample high-school project**, so file structure is simple and unorganized.
- No external frameworks are used.
- Safe for learning and demonstration purposes.
